Factorio Calculator Help page:

- Help Syntax
    []: optional arguments
    <>: mandatory arguments
     *: interactive terminal session available (run without arguments to enter)

- Commands:
    · Help: show this page
    · List [search]: show a list with all possible products (that contain the optional search)
    ·*Calc [product <amount>]: Calculate the needed subproducts for the given product and amount
                               or just use Calc to get an interactive terminal session
    ·*New  [product <time> <amount>]: Add a new product
    · Save: TODO: not implemented yet
    · Exit: exits the program
